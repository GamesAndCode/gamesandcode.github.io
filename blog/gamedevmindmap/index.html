Introduction

This post will give you a quick overview about the game dev mind map I created to organize useful web links.
You can find the mind map here .

Mind map overview

It is best to open the mind map on your computer. I tested it on mobile. It works, but feels a little off. To zoom in and out use CTRL/STRG + mouse wheel.

So far there are 7 different main categories:

  • Assets
  • Marketing
  • Funding
  • Code
  • Game Design
  • Knowledge
  • Software

I mainly use the Unity game engine. So this collection is a bit biased towards the engine, but I am sure you will still find a lot of useful resources. Some of the categories have a small description. You can open it by clicking right next to the text.
I am sure I am missing a lot of good resources, so please let me know what else I should add.
I have decided not to add books to this list because it would be too much. If you want me to recommend some game development books, let me know.

If you do not like the mind map format, you can always switch to a list format in the lower left corner. After that it looks like this:

I think it is best if you check out the mind map yourself. So thats it for this post. I hope you find some useful links!
